Push Operation

Add the element to the top of the stack.

stack.push(element); // Add element to the stack

Pop Operation

Remove and return the top element of the stack.Ensure the stack is not empty before removing.

if (!stack.empty()) {
    int element = stack.top(); // Access the top element
    stack.pop(); // Remove the top element
}

Size

Return the total number of elements currently in the stack.

int size = stack.size(); // Get the size of the stack

Empty

Check if the stack contains no elements.Return true if empty, otherwise false.

bool isEmpty = stack.empty(); // Check if the stack is empty
