Add Node at a Position in LinkedList

Steps to Add an Element at Position X

1.	Create a New Node: Initialize a new node with the given value.
2.	Check Position Validity:
	o	If X is less than 0, it is an invalid position.
	o	If X is 0, insert the new node at the beginning of the list.
3.	Traverse the List:
	o	Start from the head of the list and iterate through the nodes until you reach the position X-1.
	o	If X exceeds the length of the list, you can append the new node at the end.
4.	Insert the New Node:
	o	Adjust the pointers to insert the new node at the desired position.


						

				
ï	Time Complexity: O(n), where n is the number of nodes in the linked list.
	o	Reason: In the worst case, you may need to traverse the entire list to find the correct position to insert the new node.
