Enqueue Operation

				
Add the element to the end of the list representing the queue.

				
                

                    

                        
                        
                    

                    

                        
                        
queue.push(element); // Add element to the queue

						


                       
                    

                    
                    
                
				
            

            

                
Dequeue Operation

				
Remove and return the first element of the list representing the queue.

				
                

                    

                        
                        
                    

                    

                        
                        
if (!queue.empty()) {
    int element = queue.front(); // Access front element
    queue.pop(); // Remove front element
}

						


                       
                    

                    
                    
                
	
            

            

                
Size

				
Return the total number of elements currently in the queue.

				
                

                    

                        
                        
                    

                    

                        
                        
int size = queue.size(); // Get the size of the queue

						


                       
                    

                    
                    
                

            

			

                
Empty

				
Check if the queue contains no elements.Return true if empty, otherwise false.

				
                

                    

                        
                        
                    

                    

                        
                        
bool isEmpty = queue.empty(); // Check if the queue is empty
