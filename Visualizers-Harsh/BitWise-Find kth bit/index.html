Find Kth Binary bit

Given an integer n and a position k, find the k-th bit (0-indexed) of the integer..

1.	Start
2.	Input: Read an integer n and an integer k.
3.	Shift: Right shift n by k positions: n >> k.
4.	AND: Apply bitwise AND with 1 to extract the k-th bit: (n >> k) & 1.
5.	Output: Return or print the result.
6.	End


						

				
Time Complexity: O(1)

Here's why:
For the input values:
•	n = 13 (binary: 1101)
•	k = 2
The k-th bit is found as follows:
•	Right shift n by k: 1101 >> 2 gives 0011 (which is 3 in decimal).
•	Apply the AND operation with 1: 0011 & 0001 gives 0001.

Result: The result of the operation is 1, indicating that the 2nd bit is set.

Code : 

#include 
using namespace std;


int findKthBit(int n, int k) {
    return (n >> k) & 1;
}

int main() {
    int n = 13;  // Binary: 1101
    int k = 2;
    int result = findKthBit(n, k);
    cout << "The " << k << "-th bit of " << n << " is: " << result << endl;  // Output: 1
    return 0;
}

                        
public class FindKthBit {

   
    public static int findKthBit(int n, int k) {
        return (n >> k) & 1;
    }

    public static void main(String[] args) {
        int n = 13;  // Binary: 1101
        int k = 2;
        int result = findKthBit(n, k);
        System.out.println("The " + k + "-th bit of " + n + " is: " + result);  // Output: 1
    }
}
